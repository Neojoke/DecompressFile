# uncompresslogs tool
## Description:
    Just for uncompress log files.
## Usage:
    ```
    uncompresslogs [-h] --expression [EXPRESSION [EXPRESSION ...]] src

    positional arguments:
        src                   要解压缩的文件所在目录的路径

    optional arguments:
        -h, --help            show this help message and exit
        --expression [EXPRESSION [EXPRESSION ...]], -e [EXPRESSION [EXPRESSION ...]]
                                                                压缩文件文件匹配的表达式
    ```
